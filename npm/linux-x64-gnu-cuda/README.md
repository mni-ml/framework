# @mni-ml/framework-linux-x64-gnu-cuda

Linux x86_64 (glibc) CUDA native binary for `@mni-ml/framework`.

Requires an NVIDIA GPU with CUDA drivers installed. If no GPU is available,
install the CPU-only package instead: `npm install @mni-ml/framework`

This package is installed automatically by `@mni-ml/framework` on Linux x64 systems — you should not need to install it directly.
